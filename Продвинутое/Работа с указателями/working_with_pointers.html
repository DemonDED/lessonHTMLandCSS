Свойство user-select - управляет поведением выделения элементов на странице.

  1. none - текст не выделяется
  2. contain - выделение в пределах элемента
  3. all - все содержимое родителя с этим значением
  4. auto - вычисляемое значение, автоматически определяется следующим образом: для псевдоэлементов after и before значение равно none, для редактируемого элемента значение равно contain, если у родителя элемента значение all или none - такое же будет и у самого элемента, иначе значение будет text.
  5. text - пользователь может выделить текст в элементе.
    1.Выведите на экран какой-нибудь код так, чтобы переносы и отступы в нем стояли так, как это сделано в редакторе кода. 

    2.Модифицируйте предыдущую задачу так, чтобы слева возле каждого ряда кода стоял номер этого ряда. 

    3.Модифицируйте предыдущую задачу так, чтобы при выделении и копировании кода номера рядов не копировались.

Свойство pointer-events - позволяет убрать реакцию элемента на события мыши.

Свойство cursor - устанавливает внешний вид курсора при наведении.

Сделайте ссылку, которая не будет реагировать на наведение и нажатие.
Модифицируйте предыдущую задачу так, чтобы при наведении на ссылку курсор менял свой вид на запрещенный.
